# SWR 镜像 Tag 清理工具

自动清理华为云 SWR 镜像仓库中的旧 Tag，每个仓库只保留最新的 N 个 Tag。

## 功能特性

- 自动遍历指定组织下的所有镜像仓库
- 按更新时间排序，保留最新的 N 个 Tag
- 支持分页获取，无仓库数量限制
- 内置限流保护（600ms/次）与自动重试（最多3次）

## 前置条件

- Go 1.18+
- 华为云账号，并已开通 SWR 服务
- AK/SK 具备 SWR 的读写权限

## 配置

编辑 `main.go` 中的以下参数：

```go
ak        := "你的 AK"
sk        := "你的 SK"
regionID  := "cn-east-3"          // SWR 所在区域
projectId := "你的 Project ID"    // 控制台 -> 我的凭证 -> 项目列表
```

`main()` 函数中调整组织名和保留数量：

```go
namespace := "your-namespace"  // 组织名
keepCount := 3                 // 每个仓库保留的最新 Tag 数
```

### 如何获取 Project ID

1. 登录华为云控制台
2. 右上角点击账号名 → **我的凭证**
3. 切换到**项目列表**标签
4. 找到对应 Region（如 `cn-east-3`），复制项目 ID

### 如何获取 AK/SK

1. 登录华为云控制台
2. 右上角点击账号名 → **我的凭证**
3. 切换到**访问密钥**标签
4. 点击**新增访问密钥**，下载保存

## 限流说明

华为云 SWR API 限制为 **120次/分钟**。工具已内置以下保护机制：

| 机制 | 说明 |
|------|------|
| 删除间隔 | 每次删除后等待 600ms，实际速率约 100次/分钟 |
| 自动重试 | 遇到 429 限流时，依次等待 30s / 60s / 90s 后重试 |
| 最大重试次数 | 3次，超过后跳过该 Tag 并记录错误 |

## 注意事项

> ⚠️ **删除操作不可逆**，请在执行前确认 `keepCount` 设置正确。

- 建议先在测试组织验证后再对生产环境执行
- Tag 按 `Updated` 字段（最后更新时间）倒序排列，时间最新的优先保留
- 仓库 Tag 数量 ≤ `keepCount` 时自动跳过，不做任何操作
- 部分删除失败不会中断整体流程，失败信息会在最后汇总输出
